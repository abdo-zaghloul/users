# Users Management App

A simple React application to display and search users using JSONPlaceholder API.

## Technologies

- Vite
- React 19.2
- React Router DOM
- TypeScript
- Axios
- Tailwind CSS
- Font Awesome

## Features

 Fetch users from API
 Search across all fields
 Responsive design (Desktop table / Mobiles && Tablets collapsible)
 Loading & Error states

Data Flow
1️- useUsers Hook (fetch data)
        |
2️- HomePage Component (Data reception)
        |
3️- UsersContainer (receives data props)
